# operations — 連携・運用仕様書

このディレクトリには demo-builder で実装している外部サービス連携・運用の仕様を、
**クライアント説明用 / 営業向け**にまとめたドキュメントを格納する。

## 一覧

| 仕様書 | 対象 | 内容 |
|--------|------|------|
| [eパーク 転記管理](./epark-integration.md) | ドッグサロン | eパークから入った予約を自社システムへ取込・転記する仕組み |
| [LINE 連携](./line-integration.md) | ドッグサロン / 美容室 / エステ | LINE公式アカウントの予約候補メッセージを管理画面で集約・予約化 |

## このドキュメントの使い方

- 商談時に画面操作とあわせて見せる
- 「どこまで自動／どこから手動か」を整理して説明する
- カスタマイズの可否はクライアントと相談（標準・オプション・カスタムを区別）
